# Premium Salon & Spa Black & White Template

Reusable luxury salon/spa website template by Twinstinct Digital.

## Customize
- Change business name in `index.html`
- Replace phone number and WhatsApp link
- Replace Unsplash images with client images
- Edit services, pricing, address, and map
- Adjust colors in `css/style.css`

## Files
- `index.html`
- `css/style.css`
- `js/main.js`
